# CHANGELOG

このプロジェクトのバージョンは [セマンティックバージョニング](https://semver.org/lang/ja/)（`MAJOR.MINOR.PATCH`）に従います。
アプリ画面左上のロゴ横に現在のバージョンが表示されます。バージョン定数は `js/core.js` の `APP_VERSION` です。

## v1.5.0
- アプリ名の横にバージョン番号を表示する機能を追加

## v1.4.0
- 工程記号のドラッグ編集中に、挿入先/離脱先のグループをバッジとステータスメッセージで明示するよう改良
  - 別グループの線・記号にホバー中：移動先グループ名をバッジ表示、ステータスに「〇〇から△△へ移動します」を表示
  - 空き地にホバー中：「グループ〇〇から外れます」の予告バッジを表示
  - 起点記号／起点直後の線／自分自身の接続線にホバー中：赤い「挿入不可」バッジと理由付き警告を表示
- 空き地へのドロップ時、ラインとグループの両方から外れて単独記号として配置されるように変更（従来は元の位置に戻るのみだった）
- ドラッグ判定中に自分自身の接続線への距離判定が常にヒットしてしまうバグを修正

## v1.3.2
- ファビコンを追加（工程図記号をモチーフにしたSVGアイコン）

## v1.3.1
- README を初見でも分かりやすい内容に整理（概要・使い方・ディレクトリ構成などを追記）

## v1.3.0
- 左サイドバーの幅をタブ切り替えで変化させず統一し、ドラッグで任意の幅（200〜440px）に手動調整できる機能を追加（直近の幅はlocalStorageに保存）
- 工程経路図のカードをクリックすると、チェックボックスをクリックしたときと同じ選択トグル挙動になるよう修正

## v1.2.0
- 接続済みの工程記号をドラッグして空き地にドロップした際、線が引き伸ばされて図が崩れる問題を修正し「挿入 or 元の位置に復帰」の挙動に統一
- 記号を無選択（無所属）で新規作成した場合に、自動的に「無名グループ」へ所属させるように変更（作成後の整理をしやすくするため）

## v1.1.0
- 記号をフロー線にドラッグ＆ドロップで挿入する際、意図しない位置に挿入され複数の線が突然出現する不具合を修正
- 加工・検査記号への通し番号付与ルールを「最も長いメインライン基準」に統一し、整列・チェック後も安定して連番が振られるよう修正

## v1.0.0
- 初期リリース（工程図エディタ・部品別工程別能力表の基本機能一式）
