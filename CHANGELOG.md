# CHANGELOG

このプロジェクトのバージョンは [セマンティックバージョニング](https://semver.org/lang/ja/)（`MAJOR.MINOR.PATCH`）に従います。
アプリ画面左上のロゴ横に現在のバージョンが表示されます。バージョン定数は `js/core.js` の `APP_VERSION` です。

## v1.7.0
- メイン画面のレイアウトを情報階層に沿って再構成
  - 第1階層（ヘッダー）: アプリ全体に効く操作のみ（改善前/改善後・読込/保存/印刷/画像保存・全消去）
  - 第2階層（タブ）: ビュー切り替えタブをヘッダー直下・全幅に移動（サイドバーを含む作業空間全体を切り替える上位ナビのため）
  - 第3階層（ビュー操作ツールバー）: タブ直下に新設し、選択中のビューで行う操作（戻る/進む・ズーム・番号表示・非表示配線・配置調整モード・整列・チェック等）を集約。タブより淡い背景で従属関係を表現
  - 戻る/進むボタンはリストビューでも利用可能に（従来はチャート時のみ表示）
- リストビューのインライン工程名編集で、1文字入力するたびにフォーカスが外れて入力できない不具合を修正

## v1.6.0
- 工程・グループの複製機能を追加
  - リストビューの各工程行・グループヘッダーに「複製」ボタンを追加
  - チャートビューでは記号選択時のプロパティパネルから「工程を複製」「グループ〇〇を複製」が可能
  - 工程の複製はどの線にも繋がっていない状態で追加（ドラッグして好きな位置に挿入可能）
  - グループの複製は起点・内部の接続線を含めて丸ごと複製し、他と重ならない位置に配置（外部接続・合流は複製しない）
- 起点（内製/外製）直後などに存在する非表示配線を、作成中だけ可視化できる「非表示配線」表示モードを追加（印刷・画像保存では引き続き非表示）
- 接続線を引けない操作（循環接続・別グループへの直接接続・重複接続）を行った際、目立つトースト通知で理由を表示するように変更（従来はステータスバーの小さな文字のみで見落としやすかった）
- 選択中の工程のプロパティパネルに、所属グループをバッジで常時表示するように変更（クリックでグループ変更も可能）
- ドラッグ移動時に配線の組み替え（挿入/抜き取り）を行わず、位置の調整だけを行える「配置調整モード」を追加（レイアウトの微調整用）

## v1.5.0
- アプリ名の横にバージョン番号を表示する機能を追加

## v1.4.0
- 工程記号のドラッグ編集中に、挿入先/離脱先のグループをバッジとステータスメッセージで明示するよう改良
  - 別グループの線・記号にホバー中：移動先グループ名をバッジ表示、ステータスに「〇〇から△△へ移動します」を表示
  - 空き地にホバー中：「グループ〇〇から外れます」の予告バッジを表示
  - 起点記号／起点直後の線／自分自身の接続線にホバー中：赤い「挿入不可」バッジと理由付き警告を表示
- 空き地へのドロップ時、ラインとグループの両方から外れて単独記号として配置されるように変更（従来は元の位置に戻るのみだった）
- ドラッグ判定中に自分自身の接続線への距離判定が常にヒットしてしまうバグを修正
